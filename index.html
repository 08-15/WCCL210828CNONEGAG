欢迎使用您的新邮箱体验

让您的私人助理讲解功能


你好，我叫玛丽；作为您的新助手，让我简要介绍一下我们如何才能顺利高效地合作。
我随时为您服务，并为您提供来自未知发件人的消息的安全预览等。您可以通过主题行的格式识别这些消息，主题行总是以我的名字开头，例如：

玛丽：[info@e-mail.xing.com]

主题行始终以我的姓名开头，并在括号中包含发件人的电子邮件地址，例如：[abc@unknown.com]。

因此，我的消息很容易被发现，以向您保证这些预览是安全的，并且显示的内容没有活动功能，例如可点击的链接。
因此，您已经知道这是来自不在您的活动联系人列表中的某个人的消息。因此，您应该仔细研究它们（尤其是发件人的电子邮件地址），以决定它是一封重要的信件，还是垃圾邮件，甚至是诈骗邮件。如果您不确定并且该消息似乎不重要，则最好将其删除。同一发件人可以选择在自动阻止之前给您写另一封电子邮件（使用标准设置）。

提示：一个重要的指标是发件人电子邮件地址的格式和内容：看起来是否正常，是否与邮件内容有关？

如果您的检查结果为正，您将需要原始邮件（我已将其隔离以保护您）并希望激活发件人，以便他们的下一封电子邮件立即发送到您的收件箱。

如何激活未知发件人

请注意：在注册您的帐户时，我为您的邮箱创建了两个子文件夹：

一个称为“#Activate”，另一个称为“#Block-URL”。这两个文件夹是您的活动命令通道。

要激活新发件人，只需将关联的预览消息移动/移动到 #Aktivate 文件夹（例如通过拖放），然后等待来自发件人的原始消息到达。

如何阻止 URL

一些垃圾邮件发送者使用带有无数发件人名称的地址 URL 以不同的名称重复发送邮件。如果你有这种经历，想快速摆脱这种困扰，可以屏蔽发件人的网址。只需将此类发件人的单个预览移动到#Block-URL。已经完成了！

请注意：切勿移动预览，例如从 Gmail、Yahoo 等发件人发送到 #Block-URL，因为这将阻止此服务的所有用户。如果您在 URL 块上做错了什么，没问题。只需从#Block-URL 文件夹中删除相应的预览即可。

注意：对于 Microsoft Outlook，您必须分配这些文件夹才能使它们可见。请注意我们 YouTube 频道上的特殊说明视频！


其他功能：除了这两个直接渠道之外，您还可以通过向您自己的电子邮件地址（您自己的帐户）发送电子邮件来向我发送电子邮件以了解我的所有其他功能。 只需以所需格式在主题行中输入您的命令，如下所述。

这里每个命令都在一个单独的步骤中解释（更多的功能将在后面）。

重要提示：下面的示例显示了我的名字“Marie”，您需要将其替换为您的个人助理的姓名，该姓名将出现在您的欢迎信息中。 非常感谢。
注册命令:

将命令放在如下所示的参考行中并发送到您自己的地址

注册命令

Marie:3

在阻止发件人之前发送的预览数量（例如，3 将在阻止之前发送 3 次预览）
注意：设置 0 完全关闭自动阻止，意味着您将获得任何未知发件人消息的预览，并且不会阻止任何发件人！

How to register and activate email addresses from my email program?

• Select one of your accounts that is registered on your Spam-X eMailAgent and should have addresses added on, e.g. me@myaccount.com
• Make a "new mail" addressed to that account, in this example: me@myaccount.com
• Instruct your mail assistant (in this example it is Andre to register one or more addresses with the following command in the subject line of that mail:
Andre:register(new@contact.de, eric@malo.com)
